# FIRE Simulator

Side FIRE / Monte Carlo Simulator for early retirement planning.

**Live demo:** https://kinoko178yuzu-ux.github.io/fire-simulator/

## Features
- Monte Carlo simulation (1000 trials) for retirement failure probability
- 3-asset bucket strategy (cash / index / dividend stocks)
- Detailed tax & social insurance calculation (国民健康保険 / 任意継続 / 介護保険)
- 3 FIRE scenarios with pension start age 60-75
- Side income with end age, spouse part-time income
- Child independence cost reduction
- Planned purchases (車買い替え, 住宅修繕, etc.) with recurrence
- iDeCo / 退職金 / NISA support
